🚀 التشغيل:

الخطوة 1: تثبيت المكتبات
$ pip install -r requirements.txt

الخطوة 2: تشغيل الخادم
$ python AI.py

الخطوة 3: فتح الموقع
→ افتح index.html في المتصفح

الخطوة 4: اختبار الـ API (اختياري)
$ python test_api.py

🐛 استكشاف الأخطاء:

❌ "No module named flask"
   ✅ الحل: pip install flask flask-cors

❌ "Address already in use"
   ✅ الحل: غيّر PORT في config.py

❌ الـ Chatbot لا يظهر
   ✅ الحل: فتح Console (F12) وشوف الأخطاء

❌ "Connection refused"
   ✅ الحل: تأكد من تشغيل AI.py
